# CoffeeShop
Добрый день!
1. Решила, что требование по БД - это проверка на знание текущей обстановки. 
И могу пояснить следующее: MSSQL Express - не является продакшен решением, 
а использование коммерческих версий БД от MS является рисками и не желательно при текущей политической ситуации. 
В качестве промышленного решения я бы использовала PG, но в конкретном задании использовала sqlite, что бы облегчить вам разворачивание при просмотре задания.
2. Страница администрирования доступна по адресу /admin/{token}. 
Ключ для доступа к странице администрирования задается через переменную окружения. и в проекте задан как 12345.
